
USE CASE: ** Cancel Login on Basic Authentication Page
ID: ** UC-LOGIN-001

ACTORS:
- button
- User

PRECONDITIONS:
1. User is on the internet.
2. User has not previously authenticated on the website.

MAIN FLOW:
1. User navigates to the basic authentication page:  https://the-internet.herokuapp.com/basic_auth
2. User sees the header message "This site is asking you to sign in."
3. User presses the "Cancel" button.
4. 4. User is redirected to the browser's home page.
5. 5. User sees text "Not authorized" on the page.
6. **Alternative Flows:**
7. If the user has already authenticated on the website, they will not be prompted to sign in again.
8. If the user enters invalid credentials, they will be presented with an error message.

None
